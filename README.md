# مستخرج مشغلات M3U وخادم CORS الوسيط المدمج

هذا المشروع يوفر واجهة مستخدم احترافية لاستخراج القنوات وروابط البث من ملفات M3U بالإضافة إلى خادم وسيط (CORS Proxy Server) مدمج بلغة Node.js لتخطي حظر متصفحات الويب وتشغيل بث القنوات المباشر (HLS و MPEG-TS) الذي يعمل على VLC ولكن يفشل على المتصفح.

## 🚀 كيفية التشغيل في دقيقة واحدة

1. تأكد من تثبيت **Node.js** على جهاز الكمبيوتر الخاص بك.
2. افتح موجه الأوامر (Command Prompt أو PowerShell) في هذا المجلد:
   `C:\Users\story\.gemini\antigravity\scratch\m3u-extractor`
3. قم بتشغيل الخادم بالأمر التالي:
   ```bash
   node server.js
   ```
4. افتح المتصفح واذهب إلى العنوان:
   [http://localhost:3000](http://localhost:3000)

---

## 🛠️ كيف يعمل الخادم الوسيط للتخلص من مشاكل التشغيل؟

* **تخطي حظر المتصفحات (User-Agent):** يقوم السيرفر بطلب البث من المصدر مع تزييف المتصفح وإرسال معلومات تطابق برنامج **VLC** لكي لا يرفض خادم IPTV تشغيل الفيديو.
* **تخطي حظر الـ CORS:** يضيف السيرفر المحلي ترويسات `Access-Control-Allow-Origin: *` للاستجابة حتى يقبلها متصفحك.
* **دعم تشغيل ملفات الـ TS المباشرة:** قمنا بدمج مكتبة `mpegts.js` بالإضافة لـ `hls.js` ليتمكن المشغل داخل المتصفح من تشغيل ملفات `.ts` و `.m3u8` معاً.

---

## 💡 خيارات تشغيل البث في الأداة:

1. **تفعيل الخادم الوسيط (مفعّل تلقائياً):** يقوم بتشغيل الفيديو عن طريق البروكسي الخاص بك لتخطي الحظر. عند نسخ الرابط، سيعطيك الرابط الممرر عبر البروكسي الخاص بك.
2. **إلغاء تفعيل الخادم الوسيط:** يقوم بتشغيل البث باستخدام الرابط المباشر من ملف M3U، وهذا سيعمل فقط إذا كان خادم البث يدعم الـ CORS بشكل عام.
